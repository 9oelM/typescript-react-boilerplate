# typescript-boilerplate
My boilerplate for typical typescript app run on node.

# Stacks
- [x] typescript
- [x] node^10.15
- [x] jest
- [x] prettier
- [x] tslint
- [x] eslint
- [x] webpack
- [x] husky
- [x] forever

# Usage

## Run `webpack` and run the app
```
yarn start
```

## Watch the app with `webpack` and `forever`
```
yarn watch
```

## Lint 
```
yarn lint
```

Lint configs:
```
yarn lint:config
```
